To start the server, use
$ gcc server.c -o server
$ ./server <port-no>
example 
$ ./server 5000

if "error at bind" occurs, try using different port number.

to start the client, use
$ gcc client.c -o client
$ ./client <port-no entered in server>
example
$ ./client 5000
